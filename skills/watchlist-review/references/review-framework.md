# Watchlist Review Framework

Use this framework to turn a long list of names into a smaller list that actually deserves attention.

## What a good watchlist does

A good watchlist is not a scrapbook.

It should help the user answer:

- Which names matter now?
- Why do they matter now?
- What would I do with more research on each name?
- Which names are only background context?

## Core triage questions

For each name, ask:

- Is there a catalyst, regime condition, or thesis milestone that makes this relevant now?
- Is it tradable or investable for the user's style?
- Does it add distinct information, or is it duplicative?
- Is there enough preparation already done to justify deeper work?
- What skill should come next if the name stays on the active list?

## Typical reasons to keep a name active

- upcoming earnings or other scheduled catalyst
- sector leadership or weakness that matters to the user's book
- distinct exposure the user does not already have elsewhere
- a clear thesis or setup that is close to actionable
- macro sensitivity that makes the name important for the next session or week

## Typical reasons to demote or remove a name

- no clear "why now"
- duplicates another name or ETF already on the list
- too illiquid for the user's style
- no working thesis, no catalyst, and no next step
- too much work remaining for the current timeframe

## Active vs monitor vs remove

- `active focus`: the user should do more prep soon because the name could matter in the stated timeframe
- `monitor`: useful context, but not worth immediate deep work
- `remove or defer`: not actionable enough to justify attention right now

## Boundaries

This framework does not decide:

- whether the trade will work
- what order type to use
- how large the position should be
- whether the portfolio can absorb the risk

It only decides which names deserve scarce attention.
