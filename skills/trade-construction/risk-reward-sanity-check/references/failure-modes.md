# Common Failure Modes

- Attractive ratio with no credible path to the target
- Stop placed at an obvious level that is likely to be swept before thesis confirmation
- Target based on hope rather than structure, catalyst, or historical travel
- Stop so wide that the trade becomes low-conviction by definition
- Time horizon mismatch: intraday stop with multi-week target or vice versa
- Ignoring event risk that can break the intended structure
- Multiple targets that were not tied to a position-management plan
