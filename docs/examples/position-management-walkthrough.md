# Position Management Walkthrough

Prompt:

> Use `position-management` on this NVDA swing long. I am up about 1.8R, earnings are next week, and I need to decide whether to trim or hold.

Expected outcome:

- the skill compares the current position to the original plan
- catalyst risk is treated as part of the holding decision
- the output ends with one clear management action and the condition that would change it

This is a static skill. It works from the user's open-position context without live credentials.
