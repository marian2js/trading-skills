# Evidence Gap Check Walkthrough

Prompt:

> Use `evidence-gap-check` on this swing idea in semis and tell me what I still need to know before it deserves deeper work.

Expected outcome:

- the current evidence is separated from assumptions
- the highest-impact missing information is ranked clearly
- the output says whether the next step is deeper thesis work, event research, regime work, or simple monitoring

This is a static skill. It works from the user's current notes and reasoning without live credentials.
