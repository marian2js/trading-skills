# Post Trade Debrief Walkthrough

Prompt:

> Use `post-trade-debrief` on this closed swing trade and tell me the real lesson.

Expected outcome:

- the workflow reconstructs the original plan before judging outcome
- the output says whether the issue is single-trade or pattern-level
- the result ends with one concrete process change to test

This is a workflow skill. It routes the closed trade through the minimum necessary review checks rather than doing all analysis itself.
