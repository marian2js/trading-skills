# Journal Pattern Analyzer Walkthrough

Prompt:

> Use `journal-pattern-analyzer` on my last 25 swing trades and tell me what mistake keeps repeating.

Expected outcome:

- the sample quality is stated explicitly
- repeated strengths and repeated mistakes are separated from raw outcome noise
- the output ends with one or two concrete process changes to test

This is a static skill. It works from the user's trade log or journal notes without live credentials.
