# Thesis Validation Walkthrough

Prompt:

> Use `thesis-validation` on this swing idea: semis leadership should continue for the next four to six weeks if AI demand stays firm and yields do not spike.

Expected outcome:

- the thesis is rewritten as one clear claim
- evidence, assumption, and dependency are separated cleanly
- invalidation is stated concretely enough to decide whether the idea is ready for `risk-reward-sanity-check`

This is a static skill. It works from user reasoning and context without live credentials.
