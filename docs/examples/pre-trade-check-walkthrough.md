# Pre Trade Check Walkthrough

Prompt:

> Use `pre-trade-check` on this swing long idea in semis and tell me if it is actually ready.

Expected outcome:

- the workflow runs only the checks that matter for this case
- the output identifies the first real blocker instead of pretending everything is fine
- the result ends with a clear readiness verdict and the next action

This is a workflow skill. It routes the idea through the minimum necessary underlying skills rather than doing all analysis itself.
