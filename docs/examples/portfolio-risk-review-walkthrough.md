# Portfolio Risk Review Walkthrough

Prompt:

> Use `portfolio-risk-review` on my current book before I add more semis.

Expected outcome:

- the workflow identifies the main portfolio-level risk sources
- concentration, catalyst overlap, and regime pressure are separated clearly
- the result ends with a clear portfolio action before the user adds more risk

This is a workflow skill. It routes the portfolio through the minimum necessary risk checks rather than doing all analysis itself.
